# YTVISAnnotationTool
Get assistance with SAM2, or manually create annotations for YouTube-VIS-style datasets

## Getting Started
You will need to first install [SAM2](https://github.com/facebookresearch/sam2/) to use this tool

Run this command to use the program:
```
python main.py --root_dir <path/to/videos> --annotations <path/to/annotations>
```
